# Crossbeam Claude Skills

Ready-made [Claude](https://www.claude.com/) Skills for partnership and
go-to-market teams. Each skill brings partner overlap, co-sell context, and
ecosystem signals from [Crossbeam](https://www.crossbeam.com/) into your
revenue team's AI workflows.

## Skills

### Partnerships

| Skill | What it does |
|---|---|
| Ecosystem-Informed Account Brief | Pull partner context on any account, instantly |
| Partner Rep Activation | Turn a closed deal into a partner conversation |
| Ecosystem-Informed Account Prioritization | Find the accounts worth working on this week |
| ELG Strategy Coach | Get ELG frameworks and strategy from Crossbeam |

More personas coming soon: BDR, RevOps.

## Using a skill

Each skill is published as a `.skill` file under [`skills/`](skills/). Download
the skill you want and add it to Claude (Claude Desktop, Claude Code, or the
Claude API) to make it available in your conversations.
